# 🚀 Identity Echo Interface

A modern Streamlit application that collects user input, validates submissions, and estimates AI token usage through an interactive message transmission interface.

## ✨ Features

- 👤 Collects user's name and message
- ✅ Validates empty inputs
- 📩 Displays a personalized transmission message
- 🤖 Estimates AI token usage based on message length
- 📊 Shows character count and token analytics
- 🎈 Interactive and beginner-friendly Streamlit UI
